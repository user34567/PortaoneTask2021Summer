def main():
    files = get_files()

    names = get_unique_names(files[1], names_in_first_file)
    print_names(names)
    
Функція get_files() робить запит до користувача на введеня шляхів до 2 файлів, пеевіряє правельність введених шляхів, розміщає їх в пам'яті та повертає 2 файли (return [file1, file2]) 

Функція get_names_from_file(file) отримує файл що лежить в памяті, створює set names. Потім починає читати файл по рядках. Кожен рядок розбиває на частини, використовуючи роздільник "user: ", в результаті чого стаорюеться список з елементів(типу string) в яких перше слово ім'я користувача, за вийнятком 1-го елементу, оскільки в ньому буде все що записано перед першим ім'ям не враховуючи "user: ". Потім функція обробляє кожен елемент окрім першого, отримує з нього ім'я та вносить його до names створеного на початку. Після того як файл був повністю прочитаний функція його закриває. Оскільки файл закінчується на лапки, тоді якщо в кінці буде ім'я, то воно матиме вигляд name", тому в кінці перед return функція робить перевірку на те чи має останнє внесене ім'я лапки, і якщо має то в names ім'я name" заміняється на name. Функція повертає names.

Функція get_unique_names(file, unique_names) працює аналогічн до get_names_from_file. Вона не створює set names, а коли знаходть ім'я в файлі то видаляє це ім'я з unique_names, якщо воно там є. Повертає unique_names в якому видалені всі імена що повтрюються. 

Функція print_names(names) - вивід імен.
